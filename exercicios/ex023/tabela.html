Tabela Simples

Toda as linhas horizontais, como a primeira linha destacada (de vermelho), em HTML se chama, em inglês, TABLE ROW (tag: ≺tr≻)

O item/dado dentro da tabela, como o destado em azul na terceira linha (tr), leva o nome TABLE DATA (dado de tabela, tag: ≺td≻)

OBS: segundo a W3C as tags ≺tr≻ e ≺tr≻ não necessitam de tags de fechamento para funcionar, ficando a disposição do desenvolver optar por usá-las ou não.
